🌿 Qurbani Mart
📌 Project Purpose

Qurbani Mart is a modern e-commerce web application designed to provide users with a smooth and responsive experience for browsing and purchasing sacrificial animals.
Users can view animals, filter by category, sort by price, see detailed information, and book animals after authentication.

🚀 Live URL

🔗 https://b13-a8-qurbani-mart.vercel.app

✨ Key Features

🛍️ Browse animals (Cow, Goat, etc.)
🔍 Filter animals by category
💰 Sort animals by price (Low to High / High to Low)
📄 Dynamic animal details page
👤 User authentication (Login & Register)
🖼️ Profile management with image support
📦 Booking system with form validation
⚡ Fast and responsive UI using Next.js
📱 Mobile-friendly design
🔔 Toast notifications for user feedback
🎬 Lottie animations for smooth UI experience
🚪 Protected routes (login required for details/booking)
